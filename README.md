# lawrencelwl

Personal website built with Next.js: landing page, about, work showcase, and supporting UI (theme toggle, photos, projects).

## Requirements

- [Node.js](https://nodejs.org/) 22 or newer
- [Yarn](https://yarnpkg.com/) 1.x (see `packageManager` in `package.json`)

## Setup

```bash
yarn install
```

## Scripts

| Command       | Description              |
| ------------- | ------------------------ |
| `yarn dev`    | Start dev server         |
| `yarn build`  | Production build         |
| `yarn start`  | Run production server    |
| `yarn lint`   | Run ESLint               |

## Stack

- **Framework:** [Next.js](https://nextjs.org/) 16 (App Router)
- **UI:** React 19, [Tailwind CSS](https://tailwindcss.com/) 4, [Radix UI](https://www.radix-ui.com/), [Lucide](https://lucide.dev/) icons
- **Theming:** [next-themes](https://github.com/pacocoursey/next-themes)
- **Analytics:** [@vercel/analytics](https://vercel.com/docs/analytics)
